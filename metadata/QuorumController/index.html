    QuorumController¶

    leaderImbalanceCheckIntervalNs¶

    QuorumController is givenleaderImbalanceCheckIntervalNs when created.

    The interval is used to maybeScheduleNextBalancePartitionLeaders.

    maybeScheduleNextBalancePartitionLeaders¶

    • ControllerWriteEvent is requested to run
